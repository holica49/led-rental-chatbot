# 테스트 시나리오

## 1. 설치 서비스 테스트
- [ ] 실내 설치 → 서울 → 기업 → 견적 → 1000만원 이하 → 2025년 8월
- [ ] 실외 설치 → 경기 → 상가 → 구매 → 3000-5000만원 → 3개월 후

## 2. 렌탈 서비스 테스트
- [ ] 커피박람회 / 수원메쎄 2홀 → 실내 → 목공 설치 → 1개 LED
- [ ] LED: 6000x3000 → 600mm → 오퍼레이터 필요 → 2일
- [ ] 행사기간: 2025-07-09 ~ 2025-07-11

## 3. 멤버쉽 서비스 테스트
- [ ] 멤버코드: 001 → 메쎄이상 확인
- [ ] 나머지 플로우는 렌탈과 동일

## 4. 에지 케이스
- [ ] 무대 높이 0mm 입력
- [ ] LED 5개 입력
- [ ] 잘못된 형식 입력 시 에러 처리
